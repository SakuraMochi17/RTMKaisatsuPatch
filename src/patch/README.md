# RTM Kaisatsu Patch

RealTrainMod (RTM) 向けの改札・運賃システムパッチです。  
Minecraft 1.7.10 / Forge 10.13.4 対応。

---

## アイテム

| アイテム | 説明 |
|---|---|
| **乗車券** | 出発駅・到着駅が記録された一回限りの切符。入場改札でスタンプ、出場改札で回収される。 |
| **ICカード** | 残高をNBTに保存するプリペイドカード。改札で自動精算・残高表示。入場中の駅も記録。 |
| **定期券** | 指定区間を有効期限内なら何度でも通過できる。7日・30日・90日の3種類。途中駅も自動判定。 |
| **特急券** | 指定席券売機で購入する特急・新幹線用の切符。列車名・号車・席番・区間が記録される。指定席と自由席の2種類。 |
| **設定ツール** | ブロックを右クリックして各種設定GUIを開く。スニーク+右クリックでモデル選択。 |

---

## ブロック

### 改札機
- 設定ツールで右クリック → 駅名・モード（入場専用 / 出場専用 / 入出場兼用）を設定
- 手に乗車券・ICカード・定期券のいずれかを持って右クリックで通過

**ICカードの動作**
1. 未入場状態で入場改札 → 入場記録を書き込み
2. 入場済み状態で出場改札 → 運賃を精算して残高から差引
   - 残高不足の場合は運賃・残高・不足額を表示して拒否

**乗車券の動作**
1. 未使用で入場改札（発駅一致） → 使用済みマーク
2. 使用済みで出場改札（着駅一致） → 切符を消費

**定期券の動作**
- 発駅・着駅だけでなく、路線上の途中駅も有効区間として通過可能
- 有効期限切れは拒否

---

### 券売機
- 設定ツールで右クリック → 発駅設定
- 通常右クリックで購入GUI

**3つのタブ**
| タブ | 操作 |
|---|---|
| 切符購入 | 行き先を選んで「購入」。入場券（140円）も購入可。8件以上は↑↓ボタン・マウスホイールでスクロール。 |
| ICチャージ | 金額を選んで「チャージ」。ICカードをスロットに挿す。 |
| 定期券 | 行き先・期間（7/30/90日）を選んで「定期購入」。6件以上はスクロール対応。 |

**スロット（右パネル）**
- お金スロット × 4（2×2）: RTMのお金（硬貨・紙幣）を入れる
- ICカードスロット × 1: チャージ時に挿す
- お釣りは自動計算してインベントリに返却

**定期券の割引率**
| 期間 | 割引 |
|---|---|
| 7日 | 10% OFF |
| 30日 | 25% OFF |
| 90日 | 40% OFF |

---

### 駅管理ブロック
- 設定ツールで右クリック → 駅名を入力してネットワークに登録
- ブロックを破壊すると駅データも削除
- 「ネットワークから削除」ボタンでブロックを残したまま登録解除可
- 累計売上（その駅の券売機での販売総額）を表示
- 「売上をリセット」ボタンで累計をゼロに戻す

### 路線管理ブロック
- 設定ツールで右クリック → 路線名・初乗り運賃・距離単価・乗換料金・駅順を設定
- 複数路線の乗り換え運賃もDijkstra法で自動計算
- 停車駅の順番を↑↓ボタンで並び替え可能

---

### 列車管理ブロック
- 設定ツールで右クリック → 1ブロックにつき1列車を定義する
- 設定は3ページ構成

| ページ | 設定内容 |
|---|---|
| 基本情報 | 列車ID・列車名・種別（新幹線/特急/急行）・路線・指定席料金・自由席料金 |
| 停車駅 | 路線の全駅から停車駅をトグル選択（路線の駅順を維持） |
| 号車設定 | 号車ごとに座席数・クラス（普通/グリーン）を設定 |

- 「削除」ボタンで列車定義とその予約データを一括削除

---

### 指定席券売機
- 設定ツールで右クリック → 発駅設定（駅管理ブロックで登録済みの駅から選択）
- 通常右クリックで購入GUI（多段階選択式）

**購入フロー**
1. **種別選択**: 指定席 または 自由席
2. **行き先選択**: 券売機の設置駅から出発する列車の到達駅一覧
3. **列車選択**: 該当区間を走る列車一覧（満席は灰色で選択不可）
4. **号車選択**: （指定席のみ）残席数付きで号車を表示（満席は選択不可）
5. **確認・購入**: 列車名・区間・料金を表示。「乗車券もセットで購入」トグルあり

**購入時の動作**
- RTMのお金をプレイヤーインベントリから自動収集、おつりを返却
- 指定席は座席番号を自動割り当て
- 特急券アイテムをインベントリに追加
- セット購入時は乗車券も同時に発行

---

## 運賃計算

- **距離ベース**: 駅間のブロック距離 × 1Bあたりの単価 + 初乗り運賃
- **乗換料金**: 路線を跨ぐ場合は乗換先路線の `transferFee` を加算
- **IC割引**: IC精算は1円単位、切符は10円単位（切り上げ）
- 経路はDijkstra法で最安値ルートを自動選択

## 空席情報Webページ

MCサーバー起動と同時に **ポート 19190** でWebサーバーが自動起動します。

| URL | 内容 |
|---|---|
| `http://サーバーIP:19190/` | 空席情報HTMLページ（10秒ごと自動更新） |
| `http://サーバーIP:19190/api/seats` | 空席情報をJSON形式で取得 |

Webページには全列車の号車ごとの空席数・満席状況が色分けで表示されます。

### `/kaisatsuweb` コマンド
ゲーム内チャットで実行すると、空席情報ページへのクリック可能なリンクを表示します。  
権限レベル不要（全プレイヤーが使用可能）。
