# RTM Kaisatsu Patch

**언어 / Language:** [日本語](README.ja.md) | [English](README.md) | [中文](README.zh.md) | 한국어

**RealTrainMod (RTM)** 에 개찰구, 운임, 정기권, IC 카드 등 철도 운임 시스템을 추가하는 Minecraft 1.7.10 Forge MOD입니다.

---

## 필수 환경

다음 구성 중 하나가 필요합니다.

**구성 A: RTM + NGTLib**

| 항목 | 버전 |
|---|---|
| Minecraft | 1.7.10 |
| Forge | 10.13.4.1614 이상 |
| [RealTrainMod (RTM)](https://www.curseforge.com/minecraft/mc-mods/realtrainmod) | 1.7.10.41 이상 |
| [NGTLib](https://www.curseforge.com/minecraft/mc-mods/ngtlib) | 1.7.10.32 이상 |

**구성 B: KaizPatchX (권장)**

| 항목 | 버전 |
|---|---|
| Minecraft | 1.7.10 |
| Forge | 10.13.4.1614 이상 |
| [KaizPatchX](https://github.com/Kai-Z-JP/KaizPatchX/releases/latest) | 최신 버전 |

> **[KaizPatchX](https://github.com/Kai-Z-JP/KaizPatchX/releases/latest)** 는 RTM 및 NGTLib의 버그 수정과 기능 확장을 통합한 올인원 버전입니다. RTM + NGTLib 조합 대신 단독으로 사용할 수 있습니다.

---

## 설치 방법

1. 위의 필수 MOD를 모두 다운로드하여 `.minecraft/mods/` 폴더에 넣습니다.
2. [Releases](../../releases)에서 최신 `RTMKaisatsuPatch-*.jar`를 다운로드하여 동일한 `mods/` 폴더에 넣습니다.
3. Minecraft를 실행합니다（첫 실행 시 월드 데이터가 자동으로 생성됩니다）.

---

## 빠른 시작

```
1. 역 관리 블록 설치 → 설정 도구로 우클릭 → 역 이름 등록
2. 노선 관리 블록 설치 → 설정 도구로 우클릭 → 노선명, 운임, 역 순서 설정
3. 개찰기 설치 → 설정 도구로 우클릭 → 역 이름 및 모드 설정
4. 자동 발매기 설치 → 설정 도구로 우클릭 → 출발역 설정
5. 자동 발매기에 돈을 넣고 승차권 또는 IC 카드를 구매하여 개찰구 통과
```

자세한 절차는 **[설치 가이드](docs/setup.md)** 를 참조하세요.

## 기능 목록

### 아이템

| 아이템 | 설명 |
|---|---|
| 승차권 | 출발역과 도착역이 기록된 1회용 티켓 |
| IC 카드 | 선불 잔액 카드. 자동 정산, 승하차 기록, 회사별 디자인 지원 |
| 정기권 | 지정 구간을 유효 기간 내 무제한 통과 (7/30/90일권). 갱신 지원 |
| 회수권 | 10회분 묶음. 9회분 가격으로 구매 가능 |
| 1일 자유이용권 | 구매 당일 전 구간 무제한 승차 |
| 지정석 특급권 | 차량 번호와 좌석 번호 기록. 지정석·자유석 지원 |
| 승차역 증명서 | 무인역에서 발행하여 유인역 개찰구에서 현금 정산 |
| 설정 도구 | 각 블록의 설정 GUI를 여는 도구 |

### 블록

| 블록 | 설명 |

## 자주 묻는 질문 (발췌)

**Q. 개찰구를 통과하려 하면 오류가 발생합니다**  
→ 개찰기에 역 이름이 설정되어 있는지, 노선에 역이 등록되어 있는지 확인하세요. 자세한 내용은 [FAQ](docs/faq.md)를 참조하세요.

**Q. 자동 발매기에 목적지가 표시되지 않습니다**  
→ 노선 관리 블록에서 해당 역을 노선에 추가하세요. 목적지는 노선 데이터에서 자동으로 생성됩니다.

**Q. IC 카드로 출장할 때 "입장역이 삭제되었습니다"라고 표시됩니다**  
→ 입장 후 서버 재시작 또는 역 삭제가 발생한 경우 표시되는 메시지입니다. 자동으로 초기화되므로 다시 입장하세요.

**Q. 정기권 갱신은 어디서 하나요?**  
→ 자동 발매기 「정기권」 탭 → 「갱신」 서브 모드에서 구매할 수 있습니다. 잔여 7일 이하의 정기권이 대상입니다.

그 외는 [자주 묻는 질문](docs/faq.md)을 확인하세요.
