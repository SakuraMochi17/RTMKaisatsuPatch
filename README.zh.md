# RTM Kaisatsu Patch

**语言 / Language:** [日本語](README.ja.md) | [English](README.md) | 中文 | [한국어](README.ko.md)

为 **RealTrainMod (RTM)** 添加检票闸机、票价、定期券、IC 卡等铁路运费系统的 Minecraft 1.7.10 Forge MOD。

---

## 环境要求

需要以下配置之一。

**方案 A：RTM + NGTLib**

| 组件 | 版本 |
|---|---|
| Minecraft | 1.7.10 |
| Forge | 10.13.4.1614 以上 |
| [RealTrainMod (RTM)](https://www.curseforge.com/minecraft/mc-mods/realtrainmod) | 1.7.10.41 以上 |
| [NGTLib](https://www.curseforge.com/minecraft/mc-mods/ngtlib) | 1.7.10.32 以上 |

**方案 B：KaizPatchX（推荐）**

| 组件 | 版本 |
|---|---|
| Minecraft | 1.7.10 |
| Forge | 10.13.4.1614 以上 |
| [KaizPatchX](https://github.com/Kai-Z-JP/KaizPatchX/releases/latest) | 最新版 |

> **[KaizPatchX](https://github.com/Kai-Z-JP/KaizPatchX/releases/latest)** 是整合了 RTM 和 NGTLib 并修复 bug、扩展功能的一体化版本，可替代 RTM + NGTLib 组合使用。

---

## 安装方法

1. 下载所有必要的 MOD，放入 `.minecraft/mods/` 文件夹。
2. 从 [Releases](../../releases) 下载最新的 `RTMKaisatsuPatch-*.jar`，放入同一 `mods/` 文件夹。
3. 启动 Minecraft（首次启动时将自动生成世界数据）。

---

## 快速入门

```
1. 放置车站管理方块 → 用设置工具右键单击 → 注册车站名称
2. 放置线路管理方块 → 用设置工具右键单击 → 设置线路名、票价和站序
3. 放置检票闸机 → 用设置工具右键单击 → 设置车站名称和闸机模式
4. 放置自动售票机 → 用设置工具右键单击 → 设置出发站
5. 在自动售票机投入金币，购买车票或 IC 卡，通过闸机
```

详细步骤请参阅 **[安装指南](docs/zh/setup.md)**。

## 功能一览

### 物品

| 物品 | 说明 |
|---|---|
| 乘车券 | 记录出发站和目的站的一次性车票 |
| IC 卡 | 预付余额卡，自动扣费、乘降记录、支持公司专属设计 |
| 定期券 | 指定区间在有效期内可无限次通过（7/30/90 日券），支持续期 |
| 回数券 | 10 次乘车券，仅需支付 9 次的价格 |
| 一日通票 | 购买当日全线路无限次乘车 |
| 指定席特急券 | 记录车厢和座位号，支持指定席和自由席 |
| 乘车站证明书 | 在无人站发行，在有人站闸机进行现金结算 |
| 设置工具 | 打开各方块设置 GUI 的工具 |

### 方块

| 方块 | 说明 |
|---|---|
| 检票闸机 | 支持车票、IC 卡和定期券。可配置入站/出站模式及通过消息 |
| 自动售票机 | 销售车票、IC 充值、定期券（含续期）和回数券 |
| 补票机 | 对超出付费区间的 IC 卡和车票进行现金补票 |
| 发车信息牌 | 与时刻表联动显示发车信息，支持现实时间和游戏内时间 |
| 乘车站证明书发行机 | 为无人站发行乘车证明 |
| 简易 IC 读卡器 | 仅用于查询 IC 余额和入站状态的小型终端 |
| 车站管理方块 | 管理车站名称、坐标和销售额 |
| 线路管理方块 | 管理站序、票价和所属公司 |
| 列车管理方块 | 定义特急列车、车厢和停靠站 |
| 指定席售票机 | 销售、预约和取消特急券 |

## 常见问题（摘录）

**Q. 通过闸机时出现错误无法通行**  
→ 请确认闸机是否设置了车站名称，以及该车站是否已注册到线路中。详见 [FAQ](docs/zh/faq.md)。

**Q. 自动售票机中没有显示目的地**  
→ 请在线路管理方块中将该车站添加到线路中。目的地由线路数据自动生成。

**Q. 使用 IC 卡出站时显示"入站车站已被删除"**  
→ 这是在入站后服务器重启或车站被删除时显示的消息。将自动重置，请重新入站即可。

**Q. 在哪里续期定期券？**  
→ 自动售票机「定期券」标签页 → 「继续」子模式即可购买。剩余 7 天以内的定期券可续期。

其他问题请参阅 [常见问题](docs/zh/faq.md)。
