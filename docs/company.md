# 会社管理システム

会社を作成することで、複数の路線を一つの事業者としてまとめ、ICカードの相互利用・デザイン統一・売上管理を行えます。

---

## 会社の作成

1. **路線管理ブロック** に設定ツールで右クリック
2. 「会社」タブを開く
3. 「新規会社作成」ボタンを押す
4. 以下の項目を入力する:

| 項目 | 説明 |
|---|---|
| 会社名 | フル名称（例: 〇〇電鉄） |
| 略称 | ICカードのプレフィックス（例: SKR）最大5文字 |
| カードカラー | ICカードの表示色 |
| 初乗り運賃 | 基本運賃 |
| 加算レート | 距離あたりの追加運賃（1kmごと） |

---

## 路線を会社に所属させる

1. 路線管理ブロック → 「路線」タブ
2. 対象路線を選択 → 「会社に追加」
3. 所属させたい会社を選ぶ

路線を会社に追加すると、その路線の券売機で発行されるICカードが会社ブランドになります。

---

## メンバー管理

| ロール | 権限 |
|---|---|
| オーナー | すべての設定変更・メンバー管理 |
| 管理者 | 路線・運賃設定の変更 |
| スタッフ | 売上確認のみ |

メンバーは路線管理ブロック → 「会社」→「メンバー」タブで追加・削除できます。

> サーバー OP はすべての会社のすべての権限を持ちます。

---

## ICカードの相互利用

異なる会社の路線を乗り継ぐ場合でも、ICカード1枚で精算できます。

### 条件

- 乗り継ぐ会社同士が「相互利用協定」を結んでいること
- 協定は路線管理ブロック → 「会社」→「協定」タブで締結する

### 料金計算

相互利用時の運賃は、乗換ごとに各社の運賃を個別に計算して合計します。乗換料金が設定されている場合は加算されます。

---

## 売上管理

路線管理ブロック → 「会社」→「売上」タブで確認できます。

| 項目 | 内容 |
|---|---|
| 当日売上 | その日の収入合計 |
| 累計売上 | 全期間の収入合計 |
| 路線別内訳 | 路線ごとの売上 |

管理者は売上画面から現金を回収することもできます。

---

## 会社の削除

会社を削除すると、所属する路線の運賃設定が初期化されます。路線自体は残ります。

路線管理ブロック → 「会社」→「削除」から実行できます。操作は取り消せません。
