# ブロック詳細

---

## 改札機

**用途:** 切符・ICカード・定期券・回数券での入出場を管理するゲートブロックです。

### 設置

改札口に縦1列で設置します。複数並べることで幅広の改札になります。

### 設定項目（設定ツールで右クリック）

| 項目 | 説明 |
|---|---|
| 駅名 | この改札機が属する駅。路線データと照合して運賃を計算します |
| モード | **入場** / **出場** / **両用** |
| 対応会社 | 空欄 = すべての会社のIC対応、指定 = その会社のみ |
| 通過メッセージ | 正常通過時にチャットへ表示するメッセージ |
| ブラックリスト | 特定プレイヤーの通過を禁止する |

### 対応する乗車媒体

| 媒体 | 動作 |
|---|---|
| 乗車券 | 到着駅の改札を通ると回収される |
| ICカード | 入場時に記録、出場時に運賃精算 |
| 定期券 | 有効期限・区間内なら通過、区間外はエラー |
| 回数券 | 1回分消費して通過 |
| 一日フリーパス | 当日購入分のみ有効 |
| 特急券 | 指定列車の入出場に使用 |
| 乗車駅証明書 | 無人駅からの乗車証明。出場時に運賃精算 |

## 券売機

**用途:** 各種乗車媒体を販売する自動券売機です。

### タブ一覧

| タブ | 販売内容 |
|---|---|
| 乗車券 | 選択した行き先までの一回乗車券 |
| ICカード | 新規発行・チャージ |
| 定期券 | 新規発行（7日/30日/90日）・**継続購入** |
| 回数券 | 指定区間の10回分回数券 |
| 一日フリーパス | 当日乗り放題パス |

### 定期券の継続購入

- 「定期券」タブ → 「継続」ボタンで切り替え
- インベントリ内の残り7日以内の定期券が候補として表示される
- 現在の定期券の有効期限から延長するため、日数を無駄にしない

### 設定項目

| 項目 | 説明 |
|---|---|
| 発駅 | この券売機が設置されている駅 |
| 対応会社 | 販売するICカードの会社 |

## 乗越精算機

**用途:** ICカード・乗車券で乗り越した場合に追加運賃を精算します。

### 使い方

1. 出場駅に設置された乗越精算機に右クリック
2. ICカードまたは乗車券を手に持って「精算」を押す
3. 不足分の現金をインベントリから自動で引かれる
4. 精算済みの媒体で通常の改札を通過する

## 発車標

**用途:** 駅ホームや改札外に設置して、次の発車列車情報をリアルタイム表示します。

### 設定項目（設定ツールで右クリック）

| 項目 | 説明 |
|---|---|
| タイトル | 表示名（空欄 = 駅名） |
| 駅 | 表示する発車情報の対象駅 |
| 路線 | フィルター（空欄 = 全路線） |
| ダイヤ | 使用するタイムテーブル |
| 方向 | 両方 / 下り / 上り |
| 番線 | 表示する番線番号 |
| 表示行数 | 最大8行まで |
| **時刻モード** | **現実時刻** または **ゲーム内時刻** |

### 時刻モードについて

- **現実時刻** — サーバーが動いているマシンの実際の時刻（デフォルト）
- **ゲーム内時刻** — Minecraft の world time（0 tick = 午前6:00）

### 表示の見方

| 列 | 内容 |
|---|---|
| 時刻 | 発車時刻（HH:MM） |
| 行き先 | 終点駅名 |
| 種別 | 普通 / 急行 / 特急など |
| 列車番号 | ダイヤで設定した列車識別番号 |

## 乗車駅証明書発行機

**用途:** 無人駅での乗車時に、乗車駅・時刻が記録された証明書を発行します。

### 使い方

1. 無人駅の改札付近に設置する
2. 乗客が右クリックして証明書を受け取る
3. 有人駅の改札（**入場モード**に設定した改札機）に証明書を持って通過
4. 自動で乗車区間の運賃が精算される

### 証明書の内容

- 乗車駅名
- 発行時刻（Minecraft ワールド日付）

> 証明書は一度使用すると消去されます。紛失した場合は再発行してください。

## IC簡易リーダー

**用途:** ICカードの残高・入場状態を確認するだけの小型端末です。

### 使い方

右クリックでインベントリ内のICカードの情報を表示します。改札機は通れません。

## 駅管理ブロック

**用途:** 駅の基本情報（名前・座標・売上）を管理します。

### 設定項目

| 項目 | 説明 |
|---|---|
| 駅名 | 路線管理や改札で参照される駅の識別名 |
| 表示名 | 発車標などで表示される名前（省略可） |

### 売上確認

GUI の「売上」タブで当日・累計の売上を確認できます。管理者がお金を回収することも可能です。

## 路線管理ブロック

**用途:** ブロック 1つが会社 1つを表します。路線の定義・駅順・運賃設定に加え、会社のプロパティ・メンバー・相互利用もここで管理します。

### GUI ページ構成

| ページ | 内容 |
|---|---|
| トップ | 会社情報の確認、路線一覧、各設定ページへのボタン |
| 路線編集 | 路線ID・名前・運賃・駅順を設定 |
| 会社設定 | ID・名前・カラー・ICカード名・デフォルト運賃を設定 |
| メンバー管理 | 操作権限を持つプレイヤーを追加・削除 |
| 相互利用 | 他社とのICカード相互利用を許可・解除 |

### 初回設定の流れ

1. 設定ツールで右クリック → 「会社を設定する」ボタンが表示される
2. 会社ID・名前などを入力して「保存」→ 会社が登録され通常のトップ画面になる
3. 路線を追加していく

詳細は [会社管理システム](company.md) を参照してください。

## 列車管理ブロック

**用途:** 特急列車・指定席列車の定義を行います。

### 設定内容

- 列車名・列車番号・号車数
- 停車駅と発着時刻
- 各号車の座席数・グレード（指定席 / 自由席 / グリーン）

## 指定席券売機

**用途:** 特急券の発売・予約・キャンセルを行います。

### タブ一覧

| タブ | 機能 |
|---|---|
| 購入 | 列車・日付・号車・座席を選んで購入 |
| 予約確認 | 購入済みの特急券の内容を確認 |
| キャンセル | 購入済みの特急券を払い戻す |
