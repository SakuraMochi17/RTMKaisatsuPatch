## 检票闸机

**用途:** 使用车票、IC 卡、定期券、回数券控制入站和出站的闸机方块。

### 放置

在检票口处竖向单列放置。并排放置多个可创建更宽的闸机通道。

### 配置（用设置工具右键单击）

| 设置 | 说明 |
|---|---|
| 车站 | 该闸机所属的车站，用于与线路数据比对计算票价 |
| 模式 | **入站** / **出站** / **双向** |
| 支持公司 | 空白 = 接受所有公司的 IC 卡；指定后仅支持该公司 |
| 通过消息 | 成功通过时在聊天栏显示的消息 |
| 黑名单 | 禁止特定玩家通过 |

### 支持的乘车介质

| 介质 | 行为 |
|---|---|
| 乘车券 | 在目的地出站闸机通过时被回收 |
| IC 卡 | 入站时记录，出站时自动扣除票价 |
| 定期券 | 在有效期内且在覆盖区间内可通过，否则报错 |
| 回数券 | 每次通过消耗一次乘车次数 |
| 一日通票 | 仅购买当天有效 |
| 特急券 | 用于指定列车服务的入站/出站 |
| 乘车站证明书 | 无人站乘车证明，出站时结算票价 |

## 自动售票机

**用途:** 销售各类乘车介质的自动售票机。

### 标签页

| 标签页 | 销售内容 |
|---|---|
| 乘车券 | 前往所选目的地的单次乘车券 |
| IC 卡 | 新办卡及充值 |
| 定期券 | 新办（7日/30日/90日）及**续期** |
| 回数券 | 指定线路的10次乘车回数券 |
| 一日通票 | 当日无限次乘车通票 |

### 定期券续期

- 定期券标签页 → 「定期」→ 切换至「继续」子模式
- 库存中剩余7天以内的定期券将显示为可续期候选
- 从当前到期日起延长天数，不浪费任何有效期

### 配置

| 设置 | 说明 |
|---|---|
| 出发站 | 该售票机所在的车站 |
| 支持公司 | 在此出售的 IC 卡所属公司 |

## 补票机

**用途:** 对使用 IC 卡或乘车券超出付费区间的玩家收取额外票价。

### 使用方法

1. 在出站车站右键单击补票机
2. 手持 IC 卡或乘车券，按下「结算」
3. 不足金额将从库存中自动扣除
4. 用已结算的介质通过普通闸机

## 发车信息牌

**用途:** 放置在站台或检票口附近，实时显示即将发车的列车信息。

### 配置（用设置工具右键单击）

| 设置 | 说明 |
|---|---|
| 标题 | 显示名称（空白 = 车站名） |
| 车站 | 显示发车信息的目标车站 |
| 线路 | 筛选条件（空白 = 所有线路） |
| 时刻表 | 使用的时刻表数据集 |
| 方向 | 双向 / 下行 / 上行 |
| 站台 | 显示的站台编号 |
| 显示行数 | 最多 8 行 |
| **时刻模式** | **现实时间**或**游戏内时间** |

### 时刻模式说明

- **现实时间** — 运行服务器的机器的实际时钟（默认）
- **游戏内时间** — Minecraft 世界时间（tick 0 = 早上 6:00）

### 显示列说明

| 列 | 内容 |
|---|---|
| 时刻 | 发车时间（HH:MM） |
| 目的地 | 终点站名称 |
| 种别 | 普通 / 急行 / 特急等 |
| 列车编号 | 时刻表中设定的列车识别编号 |

## 乘车站证明书发行机 {#乘车站证明书发行机}

**用途:** 为无人站的乘客发行记录乘车站和时间的证明书。

### 使用方法

1. 放置在无人站检票口附近
2. 乘客右键单击即可领取证明书
3. 在有人站，手持证明书通过**入站模式**的闸机
4. 所乘区间的票价将自动结算

### 证明书内容

- 乘车站名称
- 发行时间（Minecraft 世界日期）

> 证明书一经使用即消除。丢失后需重新发行。

## 车站管理方块

**用途:** 管理车站基本信息（名称、坐标、销售额）。

### 配置

| 设置 | 说明 |
|---|---|
| 车站名 | 线路和闸机引用的车站标识名称 |
| 显示名 | 在发车信息牌等处显示的名称（可选） |

### 销售额查看

在 GUI 的「销售额」标签页中可查看当日及累计销售额。管理员也可在此回收资金。

## 线路管理方块

**用途:** 一个方块代表一家公司。通过该方块的 GUI 管理线路定义、站序、票价，以及公司属性（成员、IC 卡相互使用）。

### GUI 页面构成

| 页面 | 内容 |
|---|---|
| 主页 | 公司信息概览、线路列表、各设置页入口按钮 |
| 线路编辑 | 线路ID、名称、票价、站序 |
| 公司设置 | ID、名称、颜色、IC 卡名称、默认票价 |
| 成员管理 | 添加或删除拥有管理权限的玩家 |
| 相互使用 | 授权或撤销与其他公司的 IC 卡相互使用 |

### 初次设置流程

1. 用设置工具右键单击 → 出现「设置公司」按钮
2. 输入公司ID、名称等，按「保存」→ 公司注册完成，进入正常主界面
3. 依次添加线路

详情请参阅[公司管理系统](../zh/company.md)。

## 列车管理方块

**用途:** 定义特急列车和指定席列车。

### 配置内容

- 列车名称、列车编号、车厢数量
- 停靠站和发到时刻
- 各车厢的座位数和等级（指定席 / 自由席 / 绿色车厢）

## 指定席售票机

**用途:** 销售、预约和取消特急券。

### 标签页

| 标签页 | 功能 |
|---|---|
| 购买 | 选择列车、日期、车厢和座位后购买 |
| 预约确认 | 查看已购买的特急券内容 |
| 取消 | 退票已购买的特急券 |
