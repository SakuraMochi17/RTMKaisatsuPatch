## 乘车券

从购买的出发站到目的地的一次性通行凭证。

### 获取方式

在自动售票机的「乘车券」标签页购买。

### 使用方法

- 手持乘车券通过入站闸机 → 盖入站章
- 在目的地通过出站闸机 → 乘车券自动被回收

### 物品信息

- **提示文本:** 显示出发站、目的地和票价
- **超出范围:** 无法通过目的地以外的闸机

### 票价计算

乘车券票价以 **10日元为单位（四舍五入进位）** 计算。

## IC 卡

只要余额充足即可反复使用的预付费卡。

### 获取方式

在自动售票机的「IC」标签页办理新卡（可能需要支付办卡手续费）。

### 使用方法

1. 通过入站闸机 → 记录入站站和时间
2. 通过出站闸机 → 自动扣除票价

### 物品信息

- **提示文本:** 余额、入站状态及最近使用记录（最多10条）
- 余额不足时无法通过（请提前充值）
- IC 票价以 **1日元为单位（四舍五入进位）** 计算

### 充值

在自动售票机的「IC」标签页充值。金额从库存中扣除。

### 退卡

不再需要时，可通过自动售票机的「IC 退卡」选项退卡，退还剩余余额（如有押金则一并退还）。

## 定期券

在有效期内可无限次通过指定区间的通行凭证。

### 获取方式

自动售票机 → 「定期券」标签页 → 「新办」子模式 → 选择目的地和期限后购买。

### 期限种类

| 种别 | 有效期 |
|---|---|
| 7日券 | 自购买日起7天 |
| 30日券 | 自购买日起30天 |
| 90日券 | 自购买日起90天 |

### 使用方法

手持定期券通过闸机即可，入站和出站均可使用。

### 覆盖区间

- 不仅限于购买时的出发站和目的地，**途中各站也可乘降**
- 无法在指定区间以外的车站使用

### 续期

剩余7天以内的定期券可通过自动售票机「定期券」→「继续」子模式续期。从当前到期日起延长天数，不浪费任何有效期。

### 提示文本颜色

| 颜色 | 含义 |
|---|---|
| 绿色 | 剩余8天以上 |
| 黄色 | 剩余1～7天（即将到期） |
| 红色 | 已过期 |

## 回数券

指定线路的10次乘车券合集，比单独购买更划算。

### 获取方式

在自动售票机的「回数券」标签页购买。

### 使用方法

- 每次通过闸机消耗一次乘车次数
- 使用满10次后自动消除

### 价格

相当于9张普通乘车券的价格（节省1次费用）。

## 一日通票

购买当天可在发行公司所有线路上无限次乘降。

### 获取方式

自动售票机 → 「定期券」标签页 → 「全日通票」。

### 限制

- **仅当天有效**（以 Minecraft 世界日期为准）
- 仅对属于发行公司的线路有效

## 特急券

记录了指定列车班次和座位的票证。

### 获取方式

从**指定席售票机**购买。

### 种类

| 种别 | 内容 |
|---|---|
| 指定席 | 车厢和座位号固定 |
| 自由席 | 无指定车厢或座位（可坐空余座位） |
| 绿色车厢券 | 仅限绿色车厢（头等舱）使用 |

### 使用方法

手持特急券通过指定列车服务的闸机。

---

## 乘车站证明书

在无人站出发时，证明乘车站和时间的凭证。

### 获取方式

右键单击**乘车站证明书发行机**即自动发行。

### 使用方法

手持证明书通过有人站的入站闸机，所乘区间的票价将自动结算。

### 注意事项

- 一经使用即消除
- 无法在与发行公司不同的公司线路闸机处使用

## 设置工具

用于打开任意方块配置 GUI 的工具。

### 使用方法

手持设置工具，**右键单击**要配置的方块。

> 没有设置工具则无法打开任何方块的配置界面。请从创造模式标签页取出备用。
