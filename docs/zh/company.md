# 公司管理系统

**语言:** [日本語](../company.md) | [English](../en/company.md) | 中文 | [한국어](../ko/company.md)

设计理念为**一个线路管理方块 = 一家公司**。放置方块即创建一家公司，所有公司设置（名称、颜色、IC 卡名称、成员、相互使用）均通过该方块的 GUI 管理。

---

## 创建公司

1. 放置**线路管理方块**，并用设置工具右键单击
2. 按下「设置公司」按钮（仅首次需要）
3. 填写以下字段并按「保存」：

| 字段 | 说明 |
|---|---|
| 公司ID | 字母数字标识符（例：`SKR`）。保存后不可更改 |
| 公司名称 | 显示名称（例：樱花电铁） |
| 颜色 | IC 卡显示颜色（16进制，例：`FF0000`） |
| IC 卡名称 | IC 卡的品牌名称（例：`樱花IC`） |
| 默认起步价 | 新建线路时自动填入的起步票价（日元） |
| 加算费率 | 新建线路时自动填入的距离加算（日元/格） |

> 公司ID用于线路、IC 卡和相互使用的识别。请谨慎设置——保存后无法更改。

---

## 管理线路

线路的添加、编辑和删除均在线路管理方块的主界面中进行。

1. 用设置工具右键单击线路管理方块
2. 按下「+ 新建线路」或「设置画面 ->」

线路将自动关联至管理该方块的公司。

---

## 成员管理

线路管理方块 GUI → 主界面 → 「成员」按钮

| 操作 | 说明 |
|---|---|
| 添加 | 输入玩家名后按「添加」 |
| 移除 | 按下该玩家旁边的「移除」按钮 |

> 服务器 OP 可以修改所有公司的设置。

---

## IC 卡相互使用

允许使用单张 IC 卡结算跨公司线路的票价。

### 设置方法

1. 线路管理方块 GUI → 按「相互使用」
2. 选择合作公司后按「允许」

### 条件

- 只要任意一方公司授权，相互使用即生效
- 撤销授权后，该方向的换乘将无法使用 IC 卡

### 票价计算

按各公司区段分别计算票价后合计。如果线路设置了换乘费用，则额外加算。

---

## 放置多个线路管理方块时

每个方块代表一家独立的公司。可以为各公司分别发行 IC 卡，并通过相互使用设置进行联动。
