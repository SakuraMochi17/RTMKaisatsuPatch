# 회사 관리 시스템

**언어:** [日本語](../company.md) | [English](../en/company.md) | [中文](../zh/company.md) | 한국어

설계 방침은 **노선 관리 블록 1개 = 회사 1개**입니다. 블록을 배치하면 회사가 생성되며, 모든 회사 설정(이름, 색상, IC 카드 이름, 멤버, 상호 이용)은 해당 블록의 GUI에서 관리합니다.

---

## 회사 생성

1. **노선 관리 블록**을 배치하고 설정 도구로 우클릭합니다
2. 「회사 설정」 버튼을 누릅니다 (최초 1회)
3. 다음 항목을 입력하고 「저장」을 누릅니다:

| 항목 | 설명 |
|---|---|
| 회사 ID | 영문/숫자 식별자 (예: `SKR`). 저장 후 변경 불가 |
| 회사 이름 | 표시 이름 (예: 사쿠라 전철) |
| 색상 | IC 카드 표시 색상 (16진수, 예: `FF0000`) |
| IC 카드 이름 | IC 카드의 브랜드 이름 (예: `사쿠라IC`) |
| 기본 기본 요금 | 신규 노선 생성 시 자동 입력되는 기본 요금 (엔) |
| 가산 요율 | 신규 노선 생성 시 자동 입력되는 거리 가산 요금 (엔/블록) |

> 회사 ID는 노선, IC 카드, 상호 이용 식별에 사용됩니다. 저장 후에는 변경할 수 없으므로 신중하게 설정하세요.

---

## 노선 관리

노선 추가, 편집, 삭제는 모두 노선 관리 블록의 메인 화면에서 수행합니다.

1. 설정 도구로 노선 관리 블록을 우클릭합니다
2. 「+ 신규 노선 생성」또는 「설정 화면 ->」을 누릅니다

노선은 해당 블록이 관리하는 회사에 자동으로 연결됩니다.

---

## 멤버 관리

노선 관리 블록 GUI → 메인 화면 → 「멤버」 버튼

| 동작 | 설명 |
|---|---|
| 추가 | 플레이어 이름을 입력하고 「추가」를 누릅니다 |
| 제명 | 해당 플레이어 옆의 「제명」 버튼을 누릅니다 |

> 서버 OP는 모든 회사의 설정을 변경할 수 있습니다.

---

## IC 카드 상호 이용

다른 회사 노선을 환승하는 경우에도 IC 카드 1장으로 정산할 수 있습니다.

### 설정 방법

1. 노선 관리 블록 GUI → 「상호 이용」 버튼을 누릅니다
2. 상대 회사를 선택하고 「허가」를 누릅니다

### 조건

- 어느 한 회사가 허가하면 상호 이용이 활성화됩니다
- 허가를 해제하면 해당 방향의 환승이 불가능해집니다

### 요금 계산

각 회사 구간의 요금을 개별로 계산한 후 합산합니다. 노선에 환승 요금이 설정되어 있는 경우 추가로 가산됩니다.

---

## 노선 관리 블록을 여러 개 배치한 경우

각 블록이 독립된 회사를 나타냅니다. 각 회사에서 IC 카드를 발행하고, 상호 이용 설정으로 연동시킬 수 있습니다.
