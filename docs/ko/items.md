## 승차권

구매한 출발역에서 목적지까지 1회 사용할 수 있는 티켓입니다.

### 입수 방법

자동 발매기의 「승차권」 탭에서 구매합니다.

### 사용 방법

- 승차권을 들고 입장 개찰기를 통과 → 입장 도장이 찍힙니다
- 목적지의 출장 개찰기를 통과 → 승차권이 자동으로 회수됩니다

### 아이템 정보

- **툴팁:** 출발역, 목적지, 요금이 표시됩니다
- **구간 외:** 목적지 이외의 개찰기는 통과할 수 없습니다

### 요금 계산

승차권 요금은 **10엔 단위(올림)** 로 계산됩니다.

## IC 카드

잔액이 있는 한 몇 번이든 사용할 수 있는 선불 카드입니다.

### 입수 방법

자동 발매기의 「IC」 탭에서 신규 발행합니다 (발행 수수료가 부과될 수 있습니다).

### 사용 방법

1. 입장 개찰기를 통과 → 입장역과 시각이 기록됩니다
2. 출장 개찰기를 통과 → 요금이 자동으로 차감됩니다

### 아이템 정보

- **툴팁:** 잔액, 입장 상태 및 최근 이용 내역 (최대 10건)
- 잔액이 부족하면 통과할 수 없습니다 (미리 충전해 두세요)
- IC 요금은 **1엔 단위(올림)** 로 계산됩니다

### 충전

자동 발매기의 「IC」 탭에서 충전합니다. 금액은 인벤토리에서 차감됩니다.

### 반납

더 이상 필요하지 않은 경우 자동 발매기의 「IC 반납」 옵션을 통해 반납하면 남은 잔액이 환불됩니다 (보증금이 있는 경우 함께 환불됩니다).

## 정기권

유효 기간 내에 지정 구간을 무제한으로 통과할 수 있는 패스입니다.

### 입수 방법

자동 발매기 → 「정기권」 탭 → 「신규」 서브 모드 → 목적지와 기간을 선택하여 구매합니다.

### 기간 종류

| 종류 | 유효 기간 |
|---|---|
| 7일권 | 구매일로부터 7일 |
| 30일권 | 구매일로부터 30일 |
| 90일권 | 구매일로부터 90일 |

### 사용 방법

정기권을 들고 개찰기를 통과하기만 하면 됩니다. 입장과 출장 모두 사용 가능합니다.

### 구간 범위

- 구매 시의 출발역과 목적지뿐만 아니라 **경로 상의 중간역에서도 승하차 가능**
- 지정 구간 외의 역에서는 사용할 수 없습니다

### 갱신

잔여 7일 이하의 정기권은 자동 발매기 「정기권」 → 「갱신」 서브 모드에서 갱신할 수 있습니다. 현재 만료일부터 날짜가 연장되므로 유효 기간이 낭비되지 않습니다.

### 툴팁 색상

| 색상 | 의미 |
|---|---|
| 초록색 | 8일 이상 남음 |
| 노란색 | 1~7일 남음 (곧 만료) |
| 빨간색 | 만료됨 |

## 회수권

지정 구간의 10회 승차권 묶음입니다. 일반 승차권보다 저렴하게 구매할 수 있습니다.

### 입수 방법

자동 발매기의 「회수권」 탭에서 구매합니다.

### 사용 방법

- 개찰기를 통과할 때마다 1회분이 소모됩니다
- 10회 소모 후 자동으로 삭제됩니다

### 가격

일반 승차권 9장분의 가격 (1회분 무료).

## 1일 자유이용권

구매 당일에 발행 회사의 모든 노선을 무제한으로 이용할 수 있습니다.

### 입수 방법

자동 발매기 → 「정기권」 탭 → 「1일 자유이용권」.

### 제한 사항

- **구매 당일만 유효** (Minecraft 세계 날짜 기준)
- 발행 회사에 속한 노선에서만 유효합니다

## 특급권

지정 열차와 좌석이 기록된 티켓입니다.

### 입수 방법

**지정석 발매기**에서 구매합니다.

### 종류

| 종류 | 내용 |
|---|---|
| 지정석 | 차량과 좌석 번호가 고정됩니다 |
| 자유석 | 차량 및 좌석 번호 없음 (빈 좌석에 착석) |
| 그린 카 티켓 | 그린 카 (퍼스트 클래스) 전용 |

### 사용 방법

특급권을 들고 지정 열차 서비스의 개찰기를 통과합니다.

## 승차역 증명서

무인역에서 출발할 때 승차역과 시각을 증명하는 서류입니다.

### 입수 방법

**승차역 증명서 발행기**를 우클릭하면 자동으로 발행됩니다.

### 사용 방법

유인역의 입장 개찰기를 증명서를 들고 통과합니다. 탑승 구간의 요금이 자동으로 정산됩니다.

### 주의 사항

- 한 번 사용하면 소모됩니다
- 발행된 회사와 다른 회사 노선의 개찰기에서는 사용할 수 없습니다

## 설정 도구

모든 블록의 설정 GUI를 여는 도구입니다.

### 사용 방법

설정 도구를 들고 설정하고 싶은 블록을 **우클릭**합니다.

> 설정 도구 없이는 어떤 블록의 설정 화면도 열 수 없습니다. 크리에이티브 탭에서 미리 꺼내 두세요.
