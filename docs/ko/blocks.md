## 개찰기

**용도:** 승차권, IC 카드, 정기권, 회수권으로 입장 및 출장을 관리하는 게이트 블록입니다.

### 배치

개찰구에 세로 1열로 배치합니다. 여러 개를 나란히 배치하면 더 넓은 개찰구가 됩니다.

### 설정 항목 (설정 도구로 우클릭)

| 항목 | 설명 |
|---|---|
| 역 | 이 개찰기가 속하는 역. 노선 데이터와 대조하여 요금을 계산합니다 |
| 모드 | **입장** / **출장** / **양방향** |
| 지원 회사 | 공백 = 모든 회사의 IC 카드 허용; 지정 시 해당 회사만 |
| 통과 메시지 | 정상 통과 시 채팅에 표시되는 메시지 |
| 블랙리스트 | 특정 플레이어의 통과를 금지합니다 |

### 지원 승차 매체

| 매체 | 동작 |
|---|---|
| 승차권 | 목적지 출장 개찰기 통과 시 회수됨 |
| IC 카드 | 입장 시 기록, 출장 시 요금 자동 차감 |
| 정기권 | 유효 기간 내 및 커버 구간 내라면 통과, 그 외는 오류 |
| 회수권 | 통과마다 1회분 소모 |
| 1일 자유이용권 | 구매 당일만 유효 |
| 특급권 | 지정 열차 서비스의 입장/출장에 사용 |
| 승차역 증명서 | 무인역 승차 증명. 출장 시 요금 정산 |

## 자동 발매기

**용도:** 각종 승차 매체를 판매하는 자동 발매기입니다.

### 탭 목록

| 탭 | 판매 내용 |
|---|---|
| 승차권 | 선택한 목적지까지의 1회 승차권 |
| IC 카드 | 신규 발행 및 충전 |
| 정기권 | 신규 발행 (7일/30일/90일) 및 **갱신** |
| 회수권 | 지정 구간의 10회 승차 회수권 |
| 1일 자유이용권 | 당일 무제한 승차 패스 |

### 정기권 갱신

- 정기권 탭 → 「정기권」 → 「갱신」 서브 모드로 전환
- 재고에 있는 잔여 7일 이하의 정기권이 갱신 후보로 표시됩니다
- 현재 만료일부터 날짜를 연장하므로 유효 기간이 낭비되지 않습니다

### 설정 항목

| 항목 | 설명 |
|---|---|
| 출발역 | 이 발매기가 설치된 역 |
| 지원 회사 | 이곳에서 판매하는 IC 카드의 회사 |

## 정산기

**용도:** IC 카드 또는 승차권으로 목적지를 초과 탑승한 경우 추가 요금을 징수합니다.

### 사용 방법

1. 출장 역에 설치된 정산기를 우클릭합니다
2. IC 카드 또는 승차권을 들고 「정산」을 누릅니다
3. 부족한 금액이 인벤토리에서 자동으로 차감됩니다
4. 정산된 매체로 일반 개찰기를 통과합니다

## 발차 안내판 {#발차-안내판}

**용도:** 승강장이나 개찰구 근처에 설치하여 다음 발차 열차 정보를 실시간으로 표시합니다.

### 설정 항목 (설정 도구로 우클릭)

| 항목 | 설명 |
|---|---|
| 제목 | 표시 이름 (공백 = 역 이름) |
| 역 | 발차 정보를 표시할 대상 역 |
| 노선 | 필터 (공백 = 모든 노선) |
| 시간표 | 사용할 시간표 데이터셋 |
| 방향 | 양방향 / 하행 / 상행 |
| 승강장 | 표시할 승강장 번호 |
| 표시 행 수 | 최대 8행 |
| **시각 모드** | **현실 시각** 또는 **게임 내 시각** |

### 시각 모드 설명

- **현실 시각** — 서버를 실행 중인 기기의 실제 시각 (기본값)
- **게임 내 시각** — Minecraft 세계 시각 (tick 0 = 오전 6:00)

### 표시 열 설명

| 열 | 내용 |
|---|---|
| 시각 | 발차 시각 (HH:MM) |
| 목적지 | 종착역 이름 |
| 종별 | 보통 / 급행 / 특급 등 |
| 열차 번호 | 시간표에 설정된 열차 식별 번호 |

## 승차역 증명서 발행기 {#승차역-증명서-발행기}

**용도:** 무인역 승객에게 승차역과 시각이 기록된 증명서를 발행합니다.

### 사용 방법

1. 무인역 개찰구 근처에 배치합니다
2. 승객이 우클릭하여 증명서를 받습니다
3. 유인역에서 증명서를 들고 **입장 모드** 개찰기를 통과합니다
4. 탑승 구간의 요금이 자동으로 정산됩니다

### 증명서 내용

- 승차역 이름
- 발행 시각 (Minecraft 세계 날짜)

> 증명서는 처음 사용 시 소모됩니다. 분실한 경우 재발행이 필요합니다.

## IC 간이 리더기

**용도:** IC 카드 잔액과 입장 상태를 확인하는 소형 단말기입니다. 개찰기 통과에는 사용할 수 없습니다.

### 사용 방법

우클릭하면 인벤토리에 있는 IC 카드 정보가 표시됩니다.

## 역 관리 블록

**용도:** 역의 기본 정보(이름, 좌표, 매출)를 관리합니다.

### 설정 항목

| 항목 | 설명 |
|---|---|
| 역 이름 | 노선 및 개찰기에서 참조하는 역 식별 이름 |
| 표시 이름 | 발차 안내판 등에 표시되는 이름 (선택 사항) |

### 매출 확인

GUI의 「매출」 탭에서 당일 및 누적 매출을 확인할 수 있습니다. 관리자가 자금을 수금할 수도 있습니다.

---

## 노선 관리 블록

**용도:** 블록 하나가 회사 하나를 나타냅니다. 노선 정의, 역 순서, 요금 설정과 함께 회사 속성(멤버, IC 카드 상호 이용)도 이 GUI에서 관리합니다.

### GUI 페이지 구성

| 페이지 | 내용 |
|---|---|
| 메인 | 회사 정보 확인, 노선 목록, 각 설정 페이지 이동 버튼 |
| 노선 편집 | 노선 ID, 이름, 요금, 역 순서 |
| 회사 설정 | ID, 이름, 색상, IC 카드 이름, 기본 요금 |
| 멤버 관리 | 관리 권한이 있는 플레이어 추가/삭제 |
| 상호 이용 | 다른 회사와의 IC 카드 상호 이용 허가/해제 |

### 초기 설정 흐름

1. 설정 도구로 우클릭 → 「회사 설정」 버튼이 표시됩니다
2. 회사 ID, 이름 등을 입력하고 「저장」 → 회사가 등록되고 일반 메인 화면이 됩니다
3. 노선을 순서대로 추가합니다

자세한 내용은 [회사 관리 시스템](../ko/company.md)을 참조하세요.

## 열차 관리 블록

**용도:** 특급 열차 및 지정석 열차를 정의합니다.

### 설정 내용

- 열차 이름, 열차 번호, 차량 수
- 정차역 및 발착 시각
- 각 차량의 좌석 수 및 등급 (지정석 / 자유석 / 그린 카)

## 지정석 발매기

**용도:** 특급권 발매, 예약, 취소를 처리합니다.

### 탭 목록

| 탭 | 기능 |
|---|---|
| 구매 | 열차, 날짜, 차량, 좌석을 선택하여 구매 |
| 예약 확인 | 구매한 특급권 내용 확인 |
| 취소 | 구매한 특급권 환불 |
