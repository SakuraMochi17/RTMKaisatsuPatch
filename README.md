# RTM Kaisatsu Patch

RealTrainMod (RTM) 向けの改札・運賃システムパッチです。  
Minecraft 1.7.10 / Forge 10.13.4 対応。

---

## アイテム

| アイテム | 説明 |
|---|---|
| **乗車券** | 出発駅・到着駅が記録された一回限りの切符。入場改札でスタンプ、出場改札で回収される。 |
| **ICカード** | 残高をNBTに保存するプリペイドカード。改札で自動精算・残高表示。入場中の駅も記録。 |
| **定期券** | 指定区間を有効期限内なら何度でも通過できる。7日・30日・90日の3種類。 |
| **設定ツール** | ブロックを右クリックして各種設定GUIを開く。スニーク+右クリックでモデル選択。 |

---

## ブロック

### 改札機
- 設定ツールで右クリック → 駅名・モード（入場専用 / 出場専用 / 入出場兼用）を設定
- 手に乗車券・ICカード・定期券のいずれかを持って右クリックで通過

**ICカードの動作**
1. 未入場状態で入場改札 → 入場記録を書き込み
2. 入場済み状態で出場改札 → 運賃を精算して残高から差引

**乗車券の動作**
1. 未使用で入場改札（発駅一致） → 使用済みマーク
2. 使用済みで出場改札（着駅一致） → 切符を消費

**定期券の動作**
- 有効区間の駅であれば、入場・出場どちらの改札も通過可能
- 有効期限切れは拒否

---

### 券売機
- 設定ツールで右クリック → 発駅設定
- 通常右クリックで購入GUI

**3つのタブ**
| タブ | 操作 |
|---|---|
| 切符購入 | 行き先を選んで「購入」。入場券（140円）も購入可。 |
| ICチャージ | 金額を選んで「チャージ」。ICカードをスロットに挿す。 |
| 定期券 | 行き先・期間（7/30/90日）を選んで「定期購入」。 |

**スロット（右パネル）**
- お金スロット × 4（2×2）: RTMのお金（硬貨・紙幣）を入れる
- ICカードスロット × 1: チャージ時に挿す
- お釣りは自動計算してインベントリに返却

**定期券の割引率**
| 期間 | 割引 |
|---|---|
| 7日 | 10% OFF |
| 30日 | 25% OFF |
| 90日 | 40% OFF |

---

### 駅管理ブロック
- 設定ツールで右クリック → 駅名を入力してネットワークに登録
- ブロックを破壊すると駅データも削除
- 「ネットワークから削除」ボタンでブロックを残したまま登録解除可

### 路線管理ブロック
- 設定ツールで右クリック → 路線名・初乗り運賃・距離単価・乗換料金・駅順を設定
- 複数路線の乗り換え運賃もDijkstra法で自動計算

---

## 運賃計算

- **距離ベース**: 駅間のブロック距離 × 1Bあたりの単価 + 初乗り運賃
- **乗換料金**: 路線を跨ぐ場合は乗換先路線の `transferFee` を加算
- **IC割引**: IC精算は1円単位、切符は10円単位（切り上げ）
- 経路はDijkstra法で最安値ルートを自動選択
